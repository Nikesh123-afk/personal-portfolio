    Responsive Design

    Media queries and flexible layouts ensure your design works on all screen sizes.

    Explanation

    Responsive design uses media queries to apply different styles based on screen size, device orientation, and capabilities.

    • Mobile-first: Start with mobile styles, enhance for larger screens
    • Breakpoints: Common sizes (576px, 768px, 992px, 1200px)
    • Flexible units: Use %, em, rem, vw, vh instead of fixed pixels
    • Images: Make images responsive with max-width: 100%

    Media Query Examples

    Mobile First Approach
    /* Base styles (mobile) */
.container {
    width: 100%;
    padding: 1rem;
}

/* Tablet and up */
@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
}

/* Desktop and up */
@media (min-width: 1024px) {
    .container {
        width: 1000px;
    }
}
    Feature Queries
    /* Check for Grid support */
@supports (display: grid) {
    .layout {
        display: grid;
        grid-template-columns: 1fr 3fr;
    }
}

/* Fallback for older browsers */
@supports not (display: grid) {
    .layout {
        display: flex;
    }
}

    Selector Reference

    Attribute Selectors
    /* Has attribute */
[data-active] { }

/* Exact value */
[type="email"] { }

/* Contains value */
[class*="btn"] { }

/* Starts with */
[href^="https"] { }

/* Ends with */
[src$=".jpg"] { }
    Structural Pseudo-classes
    /* First and last child */
:first-child, :last-child { }

/* Nth patterns */
:nth-child(odd) { }
:nth-child(3n+1) { }

/* Only child */
:only-child { }

/* Empty elements */
:empty { }
    Pseudo-elements
    /* Add content before/after */
::before, ::after {
    content: "";
    display: block;
}

/* First line/letter */
::first-line, ::first-letter { }

/* Selection highlight */
::selection { }
