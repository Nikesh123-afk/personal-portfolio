Grid System

Bootstrap's 12-column grid system creates responsive layouts that adapt to any screen size.

Explanation

The grid system uses containers, rows, and columns to layout content:

  • Container: Centers and contains content
  • Row: Horizontal wrapper for columns
  • Column: Content blocks (col-sm, col-md, col-lg, col-xl)
  • Responsive: Different column sizes for different screens
<div class="container">
  <div class="row">
    <div class="col-md-8">Main</div>
    <div class="col-md-4">Sidebar</div>
  </div>
</div>

Customization & Theming

Bootstrap can be customized with CSS variables, Sass, or custom classes to match your brand.

CSS Custom Properties

Bootstrap 5 uses CSS variables for easy customization:

:root {
  --bs-primary: #0d6efd;
  --bs-success: #198754;
  --bs-danger: #dc3545;
}

.btn-custom {
  --bs-btn-bg: #6f42c1;
  --bs-btn-border-color: #6f42c1;
}
Annotation: CSS variables allow runtime theme changes.
Utility Classes

Bootstrap utilities provide quick styling without custom CSS:

.p-3 .m-2 .bg-primary .text-white .rounded

Best Practices

Clean Customization
  • Use CSS variables
  • Override with custom classes
  • Avoid modifying Bootstrap files
  • Document customizations
Reusability
  • Create component patterns
  • Use consistent spacing
  • Standardize color schemes
  • Build reusable templates
Accessibility
  • Use semantic components
  • Include ARIA attributes
  • Test with screen readers
  • Ensure keyboard navigation
Mobile-First
  • Start with mobile layouts
  • Use responsive utilities
  • Test on real devices
  • Optimize touch targets

Summary

Bootstrap accelerates development with its comprehensive component library, responsive grid system, and utility classes. It ensures consistent, professional designs while maintaining flexibility for customization.

Key Takeaway: Bootstrap's benefits include rapid development, consistent design systems, responsive layouts, and extensive community support, making it ideal for both prototyping and production applications.
